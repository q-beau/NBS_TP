NBP - TP - SETUP INSTRUCTIONS
=====================================
Anaconda Navigator must be installed on your computer
Install Spyder, Anaconda Prompt
Install R from https://cran.r-project.org/bin/windows/base/

STEP 1: Open Anaconda Prompt
   - Click Windows Start
   - Type "Anaconda Prompt"
   - Click to open

STEP 2: Navigate to Exercise Folder
   cd C:\Users\u230231\Desktop\TP_NatureBasedSolutions
   (Replace with your actual folder path)

STEP 3: Create Environment
   conda env create -f environment.yml
   
   Wait 3-5 minutes for installation to complete

STEP 4: Activate Environment
   conda activate NBS_TP
   IMPORTANT: The prompt should show (NBS_TP) when activated

STEP 5: Launch Spyder
   spyder

STEP 7: Open main_students.py and follow Instructions.docx

   
FOR NEXT TIME - Just run these 2 commands:
   conda activate NBS_TP
   spyder
